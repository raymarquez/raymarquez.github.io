Demystifying Home Loans

The purpose of this project is to analyze and visualize the home loan application trends using Machine Learning in the context of technologies learned from the UNCC Data Analytics Bootcamp.

We are using the USDA Rural Development Section 502 Guaranteed Loan Program for the list of Active Loans by Congressional District. We brought the data one level up for analytical presentation of active borrower characteristics aggregated at the state level.

To this end, data cleanup and preparations included corresponding aggregations and/or recalculations of relevant data types. A team of analysts executed reasonable deep-dive analysis on several data points (i.e., columns) to curate additional data as needed, and to apply supervised machine learning techniques leading to a revealing summary of findings.

A Project Roadmap is also available to demonstrate the steps taken by the team of analysts,and the process flows that were executed.
