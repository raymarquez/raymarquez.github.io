Demystifying Home Loans

The purpose of this project is to analyze and visualize the home loan application trends using Machine Learning in the context of technologies learned.

We are using the USDA Rural Development Section 502 Guaranteed Loan Program for the list of Active Loans by Congressional District. We brought the data a level up for analytical presentation of active borrower characteristics aggregated at the state level.

To this end, data cleanup and preparations included corresponding aggregations or recalculations for relevant data types. A team of analyst went after individual components to curate additional data as needed, and apply the supervised machine learning techniques to present a summary of findings and recommendations.
