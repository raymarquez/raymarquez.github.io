Demystifying Home Loans (for Rural Areas)

Objective

The purpose of this project is to analyze and visualize the home loan application trends using Machine Learning in the context of technologies learned from the UNCC Data Analytics Bootcamp.

Background

We used the USDA Rural Development Section 502 Guaranteed Loan Program for the list of Active Loans (by Congressional Districts). We brought the data one level up for analytical presentation of active borrower characteristics aggregated at the state (or U.S. Territory) level.

​

Approach

The data cleanup and preparations included aggregations and/or recalculations of relevant data types.

A team of analysts executed deep-dive analysis on several data points to curate additional datafiles if needed.

Executed the tools and techniques UNCC bootcampers have learned:

...Data cleanup | Data Validations & Verifications | Excel | HTML | CSS | Bootstrapping | Visualizations using MatPlotLib | Tableau Dashboarding | Machine Learning | Github for collabarotive code builds
